# SMPS Tech Lab Website

A complete static website built with HTML, CSS, and vanilla JavaScript.

## Structure

- **index.html** - Home page
- **about.html** - About page
- **investor.html** - IP Portfolio page
- **collaborate.html** - Collaboration opportunities page
- **careers.html** - Careers page
- **ecosystem.html** - Ecosystem page
- **insights.html** - Insights & blogs page
- **contact.html** - Contact page
- **style.css** - All styles in one file
- **script.js** - All JavaScript functionality
- **data.js** - All data (IPs, blogs, careers, etc.)

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern UI with animations
- ✅ Dynamic content rendering
- ✅ Search and filter functionality
- ✅ Animated stats counters
- ✅ Tab-based navigation
- ✅ Contact form with validation
- ✅ No backend required - pure static site

## How to Use

1. Open `index.html` in any web browser
2. Navigate through pages using the navigation menu
3. All data is stored in `data.js` and can be easily updated

## Customization

- **Colors**: Edit CSS variables in `style.css` (:root section)
- **Data**: Update `data.js` with your own content
- **Styling**: Modify `style.css` as needed
- **Functionality**: Extend `script.js` with additional features

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
